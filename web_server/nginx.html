WHAT IS A WEB SERVER

When you search for a website on your local machine, that info moves from your local amchine to a global area network. this is a HTTP GET REQUEST

The response from the global area network to your local machine is known as the response. The response contains the web page contents

the global area network sends your search qeries to a web server where it also gets the response of your search and serves it to you. from the WEB SERVER

WEB SERVERS

are computers which deliver the requested web pages. Every web server has an IP address and a Domain name.

Any computer can become a web server but there are applications needed to turn your machine to a web server. This server softwares include nginx and others

NGINX

This is a web server used for reverse proxying, caching and load balancing.

it also provides HTTP server capabilities.
it is designed for maximum performance and stability.

Configuration Settings

The core settings of NGINX are mainly configured in the nginx.conf file. it is structured into contexts.

installation

  • update vm: sudo apt-get update
  • install: sudo apt-get install nginx
  • adjust firewall: sudo ufw enable

if firewall is not installed

  • sudo apt update
  • sudo apt install ufw
  • status check: sudo ufw status
  • enable if not active: sudo ufw enable
  • if need be, add rules i.e allow ssh connections: sudo ufw allow ssh
  • check status again to ensure rules are applied: sudo ufw status
